# Telemetry Utils

Shared telemetry infrastructure and utility layer for MBM systems.

## Scope

- Riot API helpers
- Identity persistence utilities
- Telemetry normalization
- Environment-state helpers
- Replayable ingestion tooling
- Deterministic validation utilities

## Purpose

Telemetry Utils provides the shared infrastructure layer used across MBM telemetry and environment-state analysis systems.

Designed to support replayable ingestion, deterministic routing, and cross-engine interoperability.

## Status

Infrastructure layer.
Utilities and adapters under active development.
